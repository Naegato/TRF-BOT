## Environment

Copy `.env.example` to `.env` and fill in:
- `DISCORD_TOKEN` — Discord bot token

## Architecture

The project is a **Discord.js v14 bot** written in TypeScript (CommonJS, target ES2022).

- `src/index.ts` — entry point: initializes the Discord client, declares gateway intents, and logs in via `DISCORD_TOKEN`.
- `dist/` — compiled output (generated by `tsc`, gitignored).
- `eslint.config.mjs` — ESLint v9 flat config with `typescript-eslint`. Uses `.mjs` extension because the project is CommonJS (no `"type": "module"` in `package.json`).

**Runtime:** `tsx` is used for development (no compilation step). TypeScript compiles to `dist/` for production.

**Intents currently declared:** `Guilds`, `GuildMessages`, `MessageContent` — add new intents to the `Client` constructor in `src/index.ts` as features require them.
